Console Physics Engine
A 2D physics engine built for Windows console using pure C++ and standard libraries. Features real-time collision detection, gravity, friction, and interactive object physics with ASCII rendering.

Features
AABB collision detection & resolution

Real-time physics simulation

Impulse-based collision response

Console rendering with sprite support

Interactive player controls

Controls
A/D - Move left/right

Space - Jump

R - Circular motion

ESC - Exit

Built as a learning project to explore physics simulation and low-level console graphics in C++.
